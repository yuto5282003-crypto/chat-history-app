## 画面一覧（全18画面）

### S01. オンボーディング（3ステップ）

**目的**: アプリ思想をユーザーに伝え、初期設定を完了する

```
┌──────────────────────┐
│    [SLOTY ロゴ]       │
│                      │
│   ● ○ ○  (ステップ1)  │
│                      │
│  「SLOTYは"時間"を     │
│   売買するアプリです」   │
│                      │
│  [イラスト: カレンダー   │
│   とスロットカード]     │
│                      │
│   [次へ →]            │
└──────────────────────┘
```

**ステップ**:
1. コンセプト説明（時間枠が主役）
2. 安全設計の説明（本人確認・エスクロー）
3. チケット制度の説明 + 初回ボーナス20🎫付与

**UIコンポーネント**:
- `OnboardingCarousel`: スワイプ可能なカルーセル
- `StepIndicator`: ドットインジケータ（3ステップ）
- `PrimaryButton`: 「次へ」ボタン（薄紫アクセント）
- `SkipButton`: 「スキップ」テキストリンク（右上）
